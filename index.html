Windows XP are incă mai mulți utilizatori decât Windows 8, Vista și Chrome OS!

Îndrăgitul Windows XP a avut suportul încheiat în 2014 de Microsoft, ceea ce înseamnă că a fost neglijat în cadrul companiei și nu a primit aproape nicio actualizare – cu excepția cazurilor de securitate foarte grave, cum ar fi cazul ransomware-ului WannaCry. Cea mai recentă versiune a platformei care a primit încă o formă de atenție din partea companiei nu a mai vizat consumatorul mediu și a pierdut acest privilegiu în 2019.

Cu toate acestea, o cotă foarte mare din piață încă insistă asupra sistemului de operare întârziat, după cum se arată în datele din august 2020 de la NetMarketShare. Conform raportului, 1,26% din computerele active din lume folosesc Windows XP – mai mult decât Ubuntu (1,08%), Windows 8 (0,57%), Windows Vista (0,12%) și chiar și cele moderne versiuni de ChromeOS (0,42%).

Windows 10 (56,42%) și Windows 7 (26,03%) rămân cele mai populare, Windows 7 ajungând și la data de expirare acum în 2020. Mac OS X 10,15 (3,49%) este versiunea sistemului Apple cel mai bine poziționată în clasament.
